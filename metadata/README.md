# Screenshots for Raycast Store

To submit to the Raycast store, you'll need to add screenshots here:

1. **Required**: At least 1 screenshot (recommended: 2-5)
2. **Format**: PNG format
3. **Size**: Recommended 1280x800 or similar Raycast window size
4. **Content**: Show the extension in action:
   - Task list view with different statuses
   - Project view
   - Search/filter functionality
   - Task details with recurrence/priority/time tracking

## Screenshot naming convention:
- `hyperdash-1.png` - Main task list view
- `hyperdash-2.png` - Project view
- `hyperdash-3.png` - Task with metadata (priority, recurrence, etc.)

## How to create screenshots:
1. Open Raycast with the extension installed
2. Open the Task Notes or Project Notes command
3. Use Cmd+Shift+4 to take a screenshot
4. Save as PNG in this folder
