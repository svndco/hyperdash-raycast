# Screenshot Capture Guide

## Quick Steps

1. **Open Raycast** (Cmd+Space or your hotkey)
2. **Load hyperDASH** - Type "Task Notes" or "Project Notes"
3. **Position window** - Center it on your screen
4. **Take screenshot** - Cmd+Shift+4, then press Space to capture window
5. **Save here** - Save screenshots to this `metadata/` folder

## Screenshot Checklist

### Screenshot 1: Task List Overview (REQUIRED)
**Filename**: `hyperdash-1.png`

- Open "Task Notes" command
- Show tasks grouped by status:
  - In Progress (if you have any)
  - Todo
  - Up Next
  - Hold/Stuck, Waiting, Someday (if applicable)
- Make sure some tasks are visible with:
  - Due dates
  - Priorities (1-urgent, 2-high, etc.)
  - Project associations

**Tip**: This is the most important screenshot - shows the core functionality!

### Screenshot 2: Project View
**Filename**: `hyperdash-2.png`

- Open "Project Notes" command
- Show a list of your projects
- Display project metadata if available

### Screenshot 3: Task with Rich Metadata (RECOMMENDED)
**Filename**: `hyperdash-3.png`

- Open "Task Notes" command
- Focus on a task that shows:
  - Priority indicator
  - Due date
  - Recurrence pattern (if you have recurring tasks)
  - Time tracking (if enabled)
  - Project association

### Screenshot 4: Search/Filter (OPTIONAL)
**Filename**: `hyperdash-4.png`

- Type a search query in the Raycast search bar
- Show filtered results

### Screenshot 5: Action Menu (OPTIONAL)
**Filename**: `hyperdash-5.png`

- Select a task and press Cmd+K
- Show available actions

## Pro Tips

### Before Taking Screenshots

1. **Clean up your task names** - Remove any sensitive/personal info
2. **Have variety** - Mix of different statuses, priorities, due dates
3. **Use real examples** - But keep them generic/professional
4. **Check your theme** - Light or dark mode (dark mode often looks better)

### Screenshot Quality

- **Format**: PNG (automatically done with macOS screenshot)
- **Size**: Raycast window default size is perfect (usually ~1280x800)
- **Clean background**: Use a simple desktop wallpaper or blur it
- **No personal info**: Double-check no sensitive data is visible

## After Taking Screenshots

1. **Rename files** to `hyperdash-1.png`, `hyperdash-2.png`, etc.
2. **Move to metadata/** folder
3. **Verify they look good** - Open and check quality
4. **Optional**: Compress if over 1MB each (usually not needed)
